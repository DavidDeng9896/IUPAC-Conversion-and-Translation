# Source Information

本目录代码来自 STOUT V2.0（SMILES ↔ IUPAC 名称翻译器）。

## 仓库来源

- **克隆来源**: [egonw/Smiles-TO-iUpac-Translator](https://github.com/egonw/Smiles-TO-iUpac-Translator)（Kohulan 官方仓库 fork）
- **上游仓库**: [Kohulan/Smiles-TO-iUpac-Translator](https://github.com/Kohulan/Smiles-TO-iUpac-Translator)（克隆时不可用，已使用社区 fork）

## 论文信息

- **标题**: STOUT V2.0: SMILES to IUPAC name conversion using transformer models
- **作者**: Kohulan Rajan, Achim Zielesny, Christoph Steinbeck
- **期刊**: *Journal of Cheminformatics*, 16, 146 (2024)
- **DOI**: [10.1186/s13321-024-00941-x](https://doi.org/10.1186/s13321-024-00941-x)
- **Web 应用**: https://stout.decimer.ai

## 安装

```bash
pip install -e .
# 或
pip install STOUT-pypi
```
